# 🏎️ Projet : Exotics

> **Projet scolaire (NSI)** : Site vitrine interactif dédié aux voitures de sport iconiques (Youngtimers & Supercars).

Ce projet a été réalisé pour mettre en pratique les bases du développement web statique (HTML/CSS) tout en travaillant sur une thématique de passionnés : l'automobile d'exception.

---

## 🛠️ Stack & compétences

Langages      → HTML5 · CSS3

Design        → Flexbox · Google Fonts (Racing Sans One)

Outils        → VS Code · Git

Contenu       → Intégration multimédia (Images, Iframes Vidéo)

---

## 🗂️ Détails du projet

### 🏠 Architecture du site
Le site est structuré autour d'une page d'accueil (Home) servant de hub vers des fiches détaillées :
- **BMW M3 E30** : Focus sur l'homologation DTM et l'histoire du modèle.
- **Lamborghini Gallardo** : Présentation du V10 et de l'ère moderne du constructeur au taureau.
- **Renault Clio V6** : Zoom sur l'ingénierie française et le concept du moteur central arrière.

---

### 🎨 Design & Expérience Utilisateur
> Un accent particulier a été mis sur l'immersion visuelle et sonore.

- **Navigation** : Barre de navigation fixe (Navbar) pour un accès rapide aux différentes sections.
- **Mise en page** : Utilisation de **Flexbox** pour aligner proprement les aperçus des véhicules et leurs descriptions.
- **Multimédia** : Intégration de lecteurs vidéo pour permettre aux utilisateurs d'écouter la "symphonie" des moteurs (V10, 4-cylindres haute performance).
